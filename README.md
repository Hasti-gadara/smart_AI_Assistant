# 🤖 Smart AI Assistant

A modern AI-powered assistant that enables real-time conversations with intelligent responses using the Gemini API. This project demonstrates a full-stack AI application with a clean UI, fast backend, and conversational capabilities.

---

## 🚀 Features

* 💬 Real-time chat interface
* 🧠 AI-powered responses using Gemini API
* ⚡ Fast and responsive web app
* 🌐 Clean and modern UI
* 🔄 Easy setup and deployment

---

## 🧠 Project Overview

This Smart AI Assistant allows users to interact with an AI model through a web interface. It processes user input, sends it to the Gemini API, and returns intelligent responses in real time.

The project is designed to be scalable and can be extended with features like memory, voice input, and multi-language support.

---

## 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js
---


## ⚙️ Run Locally

### 🔧 Prerequisites

* Node.js installed

### 📥 Installation

```bash
npm install
```

### ▶️ Run the App

```bash
npm run dev
```

### 🌐 Open in Browser

```
http://localhost:3000
```
